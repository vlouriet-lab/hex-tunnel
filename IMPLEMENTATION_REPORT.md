## 🔍 Технические улучшения

### Паттерны добавления источников

```dart
_KeySource(
  'unique-name',              // Уникальный ID для логирования
  'https://raw.githubuser...', // Полный HTTPS URL
  KeyListType.blackList,      // Тип (black или white list)
)
```

### Безопасность

- ✅ Все URL проверяются на белый список (whitelist hosts)
- ✅ Размер ответа ограничен (5 МБ)
- ✅ Кол-во строк ограничено (120,000)
- ✅ SHA256 проверка (graceful skip если недоступна)
- ✅ Синтаксическая валидация каждого конфига
- ✅ Дедупликация по полному хешу конфига

### Производительность

- ✅ Таймауты на каждый источник (30 сек)

## 📞 Поддержка

### Если источник перестал работать:
1. Приложение автоматически переходит на следующий (fallback)
2. Пользователь не заметит (если работает ≥1 источник)
3. Разработчик удаляет источник в следующем обновлении

### Если ВСЕ источники недоступны:
1. Проверить интернет соединение
2. Использовать кэшированные конфиги (6 часов)
3. Подождать обновления источников (они обновляются каждые 9-15 минут)
