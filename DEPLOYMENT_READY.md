## Как использовать

### Для тестирования на устройстве:
```bash
# Установить APK
adb install -r build\app\outputs\flutter-apk\app-debug.apk

# Запустить приложение
adb shell am start -n com.sota.hexdecensor/.MainActivity

# В приложении перейти в Settings → Import Profiles
# Нажать "Refresh" - приложение загрузит конфиги
```

### Для релиза:
```bash
# Собрать release версию
flutter build apk --release

# APK будет в: build/app/outputs/flutter-apk/app-release.apk
```

## Поддержка

### Если источник не работает:
→ Приложение автоматически переходит на следующий источник  
→ Пользователь не заметит перебоя (если работает ≥1 источник)  
→ Разработчик обновит список в следующей версии

### Если ВСЕ источники недоступны:
→ Используется кэш (6 часов)  
→ Проверить интернет соединение  
→ Источники обновляются каждые 9-15 минут
